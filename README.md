# Dotfiles

Personal dotfiles for Arch Linux setup.

## Components
- bspwm
- sxhkd
- alacritty
- nvim
- polybar
- dunst
- picom
- rofi
- zsh
- kitty

## Installation

1. Clone the repository:
```bash
git clone https://github.com/abereg01/dotfiles.git
cd dotfiles
```

2. Run the installation script:
```bash
./scripts/install.sh
```

## Maintenance

To update package lists and push changes:
```bash
./scripts/sync.sh
```
